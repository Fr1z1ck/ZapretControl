# Инструкции по сборке Zapret GUI

## Создание исполняемого файла (.exe)

### Способ 1: Автоматическая сборка (рекомендуется)

1. **Создайте иконку (опционально):**
   ```bash
   python create_icon.py
   ```
   Это создаст файл `icon.ico` для приложения.

2. **Запустите сборку:**
   ```bash
   build.bat
   ```
   или
   ```bash
   python build_exe.py
   ```

3. **Результат:**
   Исполняемый файл будет находиться в папке `dist/ZapretGUI.exe`

### Способ 2: Ручная сборка

1. Установите зависимости для сборки:
   ```bash
   pip install -r requirements_build.txt
   ```

2. Создайте иконку (опционально):
   ```bash
   python create_icon.py
   ```

3. Соберите exe:
   ```bash
   pyinstaller --onefile --windowed --name=ZapretGUI --icon=icon.ico zapret_gui.py
   ```

## Использование готового exe

1. Скопируйте `ZapretGUI.exe` в папку с проектом Zapret (рядом с `service.bat`, `bin`, `lists`)
2. Запустите `ZapretGUI.exe` - он автоматически запросит права администратора
3. Все работает из одного файла!

## Примечания

- Иконка создается автоматически скриптом `create_icon.py`
- Если иконки нет, приложение соберется без неё
- Размер exe файла будет около 15-20 МБ (включая все зависимости)
- Файл полностью автономный - не требует установки Python
